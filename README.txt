Trustmary plugin to embed widgets and global tag.

== Installation ==

1. Upload trustmary plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add Trustmary Tag id in Trustmary settings page
